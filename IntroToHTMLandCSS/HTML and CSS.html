Display property in CSS

We have got 3 types of elements in CSS:

  1. block element:

    2. Does not matter the actual content area of an element, it always will take up the entiere line, meaning that even if there is "-" as margin value in the CSS box model (so in there is no margin value) the margin area will actually occupy the entiere line.
    EX: a paragraph by default is block element

  2. inline-block element:

    3. it takes up only the needed space, without occupying all the line. So another element could appear beside it.
    EX: image and input tags.

  3. inline element:

    4. It always appears within a line of text.
    EX: strong or u tags.
